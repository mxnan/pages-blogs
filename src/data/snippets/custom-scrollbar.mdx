# Custom Scrollbar .

### A simple paste into your globals.css

###### You can use your own properties to tinker around

> Using **Nextjs** , **TailwindCSS**

```css
@tailwind base;
@tailwind components;
@tailwind utilities;

::-webkit-scrollbar-thumb {
  @apply bg-foreground;
}
::-webkit-scrollbar-track {
  @apply bg-background;
}
::-webkit-scrollbar {
  @apply w-2 h-2 rounded-xl;

  @apply bg-transparent;
}
.dark::-webkit-scrollbar-button {
  @apply bg-secondwhite;
}
```
